# Salidas Educativas

Base inicial del portal de familias para el sistema interno del colegio.

## Requisitos

- Node.js 20+
- pnpm 10+

## Ejecutar portal de familias en local

```bash
pnpm install
pnpm dev:familias
```

Luego abrir: http://localhost:3000

## Rutas incluidas (vertical slice)

- `/` Inicio
- `/hijos`
- `/actividades`
- `/inscripciones`
- `/pagos`
- `/documentos`

## Estado

Este slice usa datos mock para validar experiencia y navegación.
