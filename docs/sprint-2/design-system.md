# Sprint 2 - Design System Base

## Tokens clave

- Brand 700: `#1f5c4d`
- Ink: `#0d1b1f`
- Paper: `#f8fbfb`
- Accent warm: ambar 200/300

## Tipografia

- Titulares: `Lora` (`font-display`)
- Cuerpo: `Plus Jakarta Sans`

## Componentes definidos

- CTA principal/secundario/ghost
- Cards de contenido y metricas
- Badges para filtros y tags de nivel
- Inputs y textarea unificados
- Tablas comparativas responsivas

## Reglas de consistencia

- CTA primario unico por viewport.
- Informacion numerica en bloques metricos destacados.
- Jerarquia de texto: etiqueta (xs), titulo (display), soporte (sm).
- Espaciado base: escala 4-6-8-10-14.
