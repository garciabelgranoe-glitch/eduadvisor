# Profile Template Spec v2

## Objetivo
Que el perfil funcione como landing de conversión confiable para familias.

## Zonas
1. Hero de identidad
- Nombre + ubicación + propuesta breve.
- Badge de estado de perfil.
- TrustStrip visible en primer scroll.

2. Bloque de evidencia
- DataEvidence: score, ratings, cuota, ubicación.
- Contexto explícito de cada métrica.

3. Decisión y acción
- CTA desktop: comparar / reseña / guardar / claim según estado.
- CTA mobile sticky: `StickyDecisionCTA`.

4. Profundidad
- Información institucional
- Reseñas recientes
- Mapa
- Galería premium
- Form de contacto (condicional premium)

## Regla trust-first
Siempre visible en la mitad superior:
- metodología,
- origen de datos,
- fecha de actualización,
- nivel de verificación.

## KPI UX
- Click-through a compare
- Submit de lead premium
- Scroll depth > 60%
