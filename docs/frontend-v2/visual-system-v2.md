# Visual System v2 - EduAdvisor

## Objetivo
Elevar percepción premium y consistencia visual en todas las superficies (discovery, profile, compare, dashboard, admin).

## Tokens v2
- Color: `--ea-v2-color-*` para fondo, superficie, borde, texto, brand, warning, danger.
- Spacing: `--ea-v2-space-*` (escala 4/8-based).
- Radius: `--ea-v2-radius-*` (sm a 2xl).
- Shadows: `--ea-v2-shadow-*` (soft/card/raised).
- Motion: `--ea-v2-motion-*` + `--ea-v2-ease-*`.

## Utilidades globales
- `.ea-surface-card`: superficie premium base.
- `.ea-transition-standard`: transición consistente.
- `.ea-enter`: entrada suave de módulos.
- `.ea-skeleton`: shimmer para estados loading.

## Reglas
- Solo 1 CTA primario por viewport.
- Contraste AA mínimo en texto y acciones.
- Componentes críticos siempre con estados `loading/empty/error/success`.
- Español consistente y copy orientado a decisión familiar.

## Fuentes
- Sans UI: `Avenir Next` stack.
- Display editorial: `Iowan Old Style` stack.
